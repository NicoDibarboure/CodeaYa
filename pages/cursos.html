Nuestros Cursos

Programador Front-End

En esta carrera de Front End aprenderás a crear un sitio web, y te sumergirás en las mejores prácticas del diseño web, trabajando con HTML y CSS.

Además, sabrás cómo programar con Javascript y React Js, lo cual te permitirá realizar sitios más complejos, dinámicos y escalables. Al finalizar, la carrera podrás desarrollarte como Front End Developer.

Programador Full-Stack

En esta carrera te desarrollarás desde cero como Full Stack Developer. Aprenderás a crear un sitio web sumergiéndote en las mejores prácticas del diseño web responsive, trabajando con HTML y CSS. Asimismo, sabrás cómo programar con Javascript, agregando interactividad y dinamismo a tus proyectos.

Por otro lado, en el curso de Programación Back End aprenderás a desarrollar aplicaciones modernas con Node.js y MongoDB. Programarás en JavaScript del lado del servidor aplicando técnicas asincrónicas, y aprenderás a trabajar con Bases de datos SQL y NoSQL.

Programador en Python

En este curso aprenderás las bases de programación uno de los lenguajes más populares en estos tiempos. Partirás desde sus fundamentos, para luego abarcar módulos y sintaxis, hasta el uso de reglas para crear tus primeras aplicaciones.

Incorporarás los conocimientos necesarios de Class-Based-View, login - register - accounts, CRUD y modelforms. Desarrollarás una aplicación web estilo blog de la mano de Python en Django Framework.
